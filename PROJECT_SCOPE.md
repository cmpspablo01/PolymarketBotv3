# Project Scope

## Goal
Build a modular research and execution framework for BTC 15m Polymarket directional binary markets.

## Phase 1 goal
Only set up the infrastructure:
- project structure
- Python environment
- config system
- structured logging
- module skeletons
- base data models
- tests
- documentation

## Not in scope for Phase 1
- no live trading
- no strategy logic
- no backtesting logic
- no machine learning training
- no reinforcement learning
- no recovery/lock/hedge mechanics yet

## Design philosophy
- modular
- auditable
- verbose logging
- easy to extend
- barebones first
- avoid hidden assumptions
- no unnecessary complexity
